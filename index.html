Introduction

Hello reader!

In case you're wondering what you're looking at, this is something of a hobby writing project of mine, that I thought would be fun to put online for the coding practice. Which I say partially hoping the site will be forgiven for any jankiness -- I am not a professional web developer, and all the code is from scratch, so though I try my best it will not be flawless by any means. Also, I have yet to optimize it fully for small screens, apologies to any mobile users.

As for the story itself, it's about about magic, monsters, a series of mysterious disappearances,and coworkers who hate each other.

Enjoy!
